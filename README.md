# Vue PWA 
> Comes with auth, simple mongodb, serviceworker, and vuex.

##Client Setup
``` bash
# install dependencies
cd client
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

##Server Setup
``` bash
# install dependencies
cd server
npm install

# run server at localhost:3000
npm start
```
